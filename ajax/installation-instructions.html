Installation Instructions - Super AJAX Programming Seed v.1.0

We made installing the Super AJAX Programming Seed. Here's what you need to do to add it to your web site:

  1. Download and unzip Super AJAX Programming Seed from www.ImpliedByDesign.com (which you've already done)
  2. Upload the entire folder to your web site, or add it to a local server.
  3. There are two ways you can use the script. 1) Use it to as a seed script for a more complex AJAX application, or 2) Add it to your public web site to illustrate how AJAX works. Either way, you install it the same, but you will want to add links to the page if you want your visitors to use it.
  4. The script contains a link with some additional text, that says something like:Super AJAX Programming Seed is a web site script offered by Implied By Design. As part of the license agreement, you may not remove this link, though you can change its placement on the page.
  5. To use it, just visit the ajaxseed/ folder.

Viola! You're done! If you have any questions or suggestions, please feel free to contact us at info@impliedbydesign.com.
